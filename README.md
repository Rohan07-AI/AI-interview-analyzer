# 🤖 AI Interview Analyzer

An AI-powered web application that analyzes interview responses and provides intelligent feedback, scoring, and performance evaluation.

🔗 **Live Demo:** https://ai-interview-analyzer-2-ik8l.onrender.com/  
🔗 **GitHub Repo:** https://github.com/Rohan07-AI/ai-interview-analyzer  

---

## 🚀 Features

- 🧠 **AI-Based Analysis**
  - Uses NLP to analyze user responses

- 📊 **Score System**
  - Generates a score out of 100 based on content quality

- 💬 **Smart Feedback**
  - Suggests improvements in communication and structure

- 📈 **Performance Evaluation**
  - Classifies responses as:
    - Excellent
    - Good
    - Needs Improvement

- 🔍 **Keyword Detection**
  - Evaluates presence of important interview-related terms

- 🎯 **Multiple Interview Questions**
  - User can select different types of interview scenarios

## ⚙️ Installation & Setup

1. Clone the repository: git clone https://github.com/Rohan07-AI/ai-interview-analyzer.git

2. Navigate to project folder:cd ai-interview-analyzer

3. Install dependencies:pip install -r requirements.txt

4. Run the application:python app.py

5. Open in browser: http://127.0.0.1:5000/

---

## 🧠 How It Works

1. User selects an interview question  
2. Enters their answer  
3. System analyzes:
   - Sentiment (positive/negative/neutral)
   - Keyword usage
   - Confidence level  
4. Generates:
   - Score (0–100)
   - Performance rating
   - Personalized feedback
